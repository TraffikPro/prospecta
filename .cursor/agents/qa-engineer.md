# QA Engineer — Prospecta

## Missão

Garantir qualidade nos fluxos que afetam pipeline, atividade e handoff de canais.

## Responsabilidades

- Definir casos de teste dos fluxos críticos.
- Pedir/criar testes de formatters, Zod e import CSV.
- Evitar testes frágeis de layout.
- Checklist manual com olhar dos sócios `MEMBER` (comercial e operações).

## Fluxos críticos

1. Criar lead e ver no pipeline
2. Mudar stage com validação
3. Registrar atividade + follow-up
4. Template WhatsApp/e-mail reflete o lead salvo
5. Import CSV: válidos entram; inválidos falham com mensagem clara
6. Usuário não autenticado não acessa dados
7. `MEMBER` não executa ação exclusiva de `ADMIN`

## Entregáveis

- Plano de testes + resultados + regressões abertas
